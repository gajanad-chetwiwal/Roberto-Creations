# Roberto Creations — Company Website

Marketing website for **ROBERTO CREATIONS LLC**, an e-commerce growth studio in Poland, Ohio.
Founder: Roberto Vazquez Villanueva.

## Stack

Static HTML/CSS/JS — no build step. Fonts: Fraunces + Inter (Google Fonts).

## Pages

- `index.html` — main site (hero, services, AI suite, results, testimonials, process, trust, about, FAQ, contact)
- `privacy.html` / `terms.html` — legal pages
- `404.html` — not-found page
- `logo.svg` — brand mark

## Local development

```sh
python3 -m http.server 4173
```

## Deploy

Deployed on Vercel. `vercel.json` enables clean URLs and security headers.

```sh
vercel --prod
```
